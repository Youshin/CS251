To compile :

SAP.java:
Ex) javac -classpath .:stdlib.jar:algs4.jar SAP.java
WordNet.java
Ex) javac -classpath .:stdlib.jar:algs4.jar WordNet.java
compiling WordNet.java will result :
Note: WordNet.java uses unchecked or unsafe operations.
Note: Recompile with -Xlint:unchecked for details.
But it is ok to run.
To run :
SAP:
Ex) java -classpath .:stdlib.jar SAP digraph1.txt digraph1.input
WordNet:
Ex) java -classpath .:stdlib.jar:algs4.jar WordNet synsets.txt hypernyms.txt wordnet.iqnput
